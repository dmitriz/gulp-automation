Live Reload Demo?

This is your dead simple file to enjoy Live Reload!

To make it work correctly:

  1. Download and Install Git, see Github Guide to Installing Git, Atlassian Guide for Git and Git for beginners: The definitive practical guide for more information.
  2. Download and Install Node.js, see How do I get started with Node.js and Fixing npm permissions for more information.
  3. Copy (clone) our Gulp Automation Repository into your local directory by typing in your terminal window: 
    git clone https://github.com/dmitriz/gulp-automation.git
    This should create the local subdirectory gulp-automation. Change to it: 
    cd gulp-automation
  4. That's it! Now start your Local server, LiveReload and launch your Browser pointing to right direction - all with one simple command: 
    npm start
    and don't worry when it "jumps" to your Browser - we are trying hard to save you from manual work ;)

Voila!

  1. Your server should start and serve the local file index.html right in your browser!
  2. Now try to modify and save it and enjoy your browser automatically reloading the changes! Your server keeps running in your terminal window and keep informing you about which files are reloaded! Finally you are the boss of your machine! ;-)

Trouble shooting:

  1. If the reload does not work, try to re-save index.html and manually reload your browser. It will (hopefully) reload itself from now on. ;-)
  2. Make sure no other server is trying to serve the same URL!
  3. Not really a trouble, but if you are wondering how to stop your server, Ctrl-C should do it.

More explanations:

  • Right now you are using the script start defined inside package.json, that runs your local gulp. To use Gulp.js's full power, install it globally (and be patient ;-): 
    npm install --global gulp
    Now launch your Server, LiveReload and your Browser even simpler with one short command: 
    gulp
